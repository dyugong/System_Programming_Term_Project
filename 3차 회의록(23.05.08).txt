3차 회의록(23.05.08)

Progress
1. arglist를 통해 argv로 받아온 명령어 혹은 실행파일을 저장
2. 받아온 명령어를 실행(백그라운드)
3. 실행한 명령어에 대한 PID 및 process 정보(CPU 사용량, MEM 사용량, system call 등) 받아오고 추출
4. 추출한 정보를 가시화

해야할 것
1. waitpid 3번째 인자(status)를 통해  child process의 PID 받아와서 정보 어떻게 받아오고 어떻게 추출할지
2. Proc file system에서 정보를 어떻게 받아올수 있는지
ㄴprocess가 종료된 PID에 대한 정보가 담겨있는지
ㄴ담겨있다면 무엇이 담겨있고 어떻게 추출할지
4. cpu%에 대한 평균을 낼수 있나
5. fork()하고난 후 exec함수를 통해 명령어 실행후  부모로 돌아와 waitpid 이후로 실행
6. getrusage함수로 자식process에 대한 정보를 구조체에 저장
