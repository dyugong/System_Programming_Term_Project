waitpid()등을 이용해 자식 프로세스의 pid정보를 읽어옴.
읽어온 pid를 바탕으로 /proc에서 해당 프로세스의 정보를 추적

위치는 아래와 같음

/proc/[pid]/status -> 실행중인 프로세스의 상태 정보를 제공합니다.

/proc/[pid]/stat -> 실행중인 프로세스의 상태 정보를 제공합니다.

/proc/[pid]/cmdline -> 실행중인 프로세스의 명령어 라인 인수를 제공합니다.

/proc/[pid]/environ -> 실행중인 프로세스의 환경 변수를 제공합니다.

/proc/[pid]/fd -> 실행중인 프로세스의 파일 디스크립터를 나열합니다.

/proc/[pid]/cpuinfo -> 실행중인 프로세스와 관련된 CPU 정보를 제공합니다.

/proc/[pid]/mem -> 실행중인 프로세스의 메모리 정보를 제공합니다.

/proc/[pid]/limits -> 실행중인 프로세스의 제한 사항 정보를 제공합니다.

/proc/[pid]/cgroup -> 실행중인 프로세스가 속한 cgroup 정보를 제공합니다.

/proc/[pid]/mounts -> 실행중인 프로세스가 사용하는 파일 시스템 정보를 제공합니다.
