<h1>Profiling Tool(Strace & Pidstat)</h1>
<ul>
  <h2><li>Strace</li></h2>
    <h4>strace -c 명령어를 구현</h4>
    <h4>실행 파일의 system call의 호출 횟수와 걸린 시간에 대한 정보</h4>
  <h2><li>Pidstat</li></h2>
    <h4>pidstat -l & pidstat -r 명령어를 구현</h4>
    <h4>실행 파일의 Cpu 와 Memory 사용량에 대한 정보</h4>
</ul>
